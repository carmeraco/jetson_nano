# Default settings of JetPack 4.2.1 for Jetson Nano

## What is this ?

This is default settings just after installing JetPack 4.2.1 for Jetson Nano.

| filename    | command           |
|-------------|-------------------|
| cpuinfo.txt | cat /proc/cpuinfo |
| meminfo.txt | cat /proc/meminfo |
| df.txt      | df                |
| dpkg.txt    | dpkg -l           |
| lspci.txt   | lspci             |
| uname.txt   | uname -a          |
